# Question 3: Node.js REST API

## Prerequisites

- Node 12+
- yarn 1.22+

Run `yarn install`.

## Start

Run `yarn start`.

The endpoints can be tested with Postman for example.

## Test

Run `yarn test`.

To see the test coverage, run `yarn test:coverage`.

To run only the functional tests, run `yarn test:func`.
